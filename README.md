# Nuance - Excalidraw Pen Sound

Satisfying paper writing sounds for digital handwriting on Excalidraw canvases in Obsidian.

This is the official free standalone sound module of the **Nuance** productivity and performance ecosystem.

---

## 🌟 Features

*   **Acoustic Feedback:** Plays real-time scratching sounds as you draw on any Excalidraw canvas.
*   **3 Sound Profiles:** Choose between a Pencil, Fountain Pen, or Marker.
*   **Custom Settings:** Adjust sound volume and tone pitch directly in the plugin settings.
*   **Preview Mode:** Listen to sound profiles inside the Settings panel without needing a stylus.

---

## 🚀 Installation

### Option 1: Obsidian Community Store (Recommended - Coming Soon)
1. Open Obsidian Settings.
2. Go to **Community plugins** -> **Browse**.
3. Search for `Nuance - Excalidraw Pen Sound` and click **Install**.
4. Enable the plugin.

### Option 2: Via BRAT (For Early Access)
1. Install the [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat) from the Obsidian Community Store.
2. Open Command Palette (`Cmd+P`) and select `BRAT: Add a beta plugin for testing`.
3. Paste this repository URL: `https://github.com/theanh97/nuance-excalidraw-pen-sound`
4. Click **Add Plugin** and enable it in your Community plugins list.

### Option 3: Manual Installation
1. Go to the [Releases](https://github.com/theanh97/nuance-excalidraw-pen-sound/releases) section and download the latest zip file.
2. Extract the folder into your vault's `.obsidian/plugins/` directory.
3. Reload Obsidian and enable the plugin in Settings.
